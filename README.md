# Artificial Intelligence-Based Healthcare Infrastructure Modeling

##  Overview
This repository contains the research and methodology for predicting and optimizing rural healthcare infrastructure in India using machine learning algorithms. The objective of this study was to predict the required number of healthcare sub-centres across various states based on existing infrastructure and medical personnel distribution.

**Note on Data Privacy:** The raw datasets and source code for this project are currently withheld due to data privacy constraints and non-disclosure agreements with the American Association of Precision Medicine. However, the complete methodology, implementation details, and evaluation metrics are thoroughly documented in the attached research paper: `The utilization of artificial intelligence in improvising the rural health infrastructure of India chicago style.pdf`.

##  Methodology
The project evaluated three distinct machine learning approaches to determine the most stable and accurate predictive model:
* Linear Regression
* Random Forest Classifier
* Artificial Neural Networks (ANN)

**Tools & Libraries Used:** Python, Pandas, NumPy, Scikit-learn.

##  Key Results
The models were evaluated using Mean Squared Error (MSE) to assess the average squared difference between observed and predicted values. The percentage difference between training and testing data demonstrated high stability across the models:
* **Neural Network:** 0.005% difference
* **Linear Regression:** 0.04% difference
* **Random Forest:** 0.05% difference
